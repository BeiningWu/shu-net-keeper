# 更新日志

所有重要的项目变更都会记录在此文件中。

本项目遵循 [语义化版本](https://semver.org/lang/zh-CN/) 规范。

## [0.1.0] - 2026-02-02

### ⚠️ 重要提示
**这是项目的初始版本，尚未进行实机测试，可能存在未知问题。请谨慎在生产环境中使用。**

### 新增
- 🎉 初始版本发布
- 🔄 校园网自动登录功能
  - 自动检测网络连接状态
  - 断网时自动重新登录
  - 可配置检查间隔时间
- 📧 SMTP 邮件通知支持
  - 登录成功后发送邮件通知
  - IP 地址变化时发送警告邮件
  - 支持常见邮箱（QQ、163、Gmail、Outlook）
  - 统一使用 465 端口（SSL）
- 🌐 IP 地址监控
  - 实时检测 IP 地址变化
  - 变更时自动发送通知
- 📝 完整的日志系统
  - 基于 tracing 的结构化日志
  - 支持文件输出和按日期归档
  - 多级别日志（INFO、WARN、ERROR、DEBUG）
- ⚙️ TOML 配置文件支持
  - 简洁的配置格式
  - 完整的配置验证
  - 详细的配置示例
- 🐳 Docker 容器化部署
  - 提供 Dockerfile 和 docker-compose.yml
  - 支持一键部署
- 📚 完整的部署文档
  - Windows 服务部署（NSSM、任务计划程序）
  - Linux systemd 服务部署
  - macOS launchd 服务部署
  - Docker 部署指南

### 技术架构
- 使用 Rust 语言开发，确保高性能和内存安全
- 模块化架构设计，代码结构清晰
- 统一的错误处理机制，便于调试和维护
- HTTP 客户端工厂模式，支持灵活配置
- 完整的单元测试框架

### 已知限制
- 未经实机测试，可能存在兼容性问题
- SMTP 功能仅在 465 端口测试过配置
- 某些邮箱服务可能需要特殊配置
